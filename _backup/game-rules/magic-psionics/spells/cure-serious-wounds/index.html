Conjuration (Healing)

Level: Brd 3, Clr 3, Drd 4, Pal 4, Rgr 4

Cleric Domains: Healing 3

This spell functions like cure light wounds, except that it cures 3d8 points of damage +1 point per caster level (maximum +15).
