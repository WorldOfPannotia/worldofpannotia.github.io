Note: This is also for adding class levels.

  1. Select monster
  2. Convert stat block to character sheet
  3. Refigure ability scores using spreadsheet & standard array (10's and 11's)
    1. Add +1 to all stats at 2, 6, 10, etc HD
      1. Creatures that cannot have Int higher than 2 retain this restriction
  4. Refigure skills
    1. Reverse engineer current skill points / synergy bonuses / ability bonuses
    2. Add skill points for increased Int
    3. Refigure skill synergies
  5. Reverse engineer feat order
  6. Add HD, template(s), and/or class levels as appropriate or desired
  7. Refigure HP
    1. Determine HP based on new Con mod and Con score
  8. Convert character sheet to new stat block
  9. Save & upload character sheet (ODS, PDF), abilities spreadsheet (ODS), stat block (DOC, PDF)
