  1. Choose race / gender
  2. Determine total levels and order
    1. ECL 40-50 => Divine Rank 0-5
    2. ECL 50-60 => DR 6-10
    3. ECL 60-70 => DR 11-15
    4. ECL 70+ => DR 16+
  3. Abilities
    1. 60 Point buy (Deity) / 42 Point buy (NPC)
    2. Based on levels and order, choose stat increases
  4. Skills
    1. Determine skill points using Excel worksheet for each set of class/racial levels.
      1. i.e. â€“ Sor 10/Wiz 10/Cle10 would have x skill points for Sorcerer skills, y skill points for Wizard skills, and z skill points for Cleric skills.
    2. Assign skill points for each set of class/racial levels.
    3. Determine synergy bonuses
  5. Determine languages
  6. Determine class abilities
  7. Choose feats
  8. Choose spells/powers known (if applicable)
  9. Choose divine abilities (if applicable)
    1. Write up divine characteristics
  10. Determine total gp
  11. Purchase equipment
  12. Determine hit points
    1. HP at first level = Con + Normal HP (max + Con mod)
    2. Average HP per die (d4 = 2.5, d6 = 3.5, d8 = 4.5, etc) rounded down if total includes .5
  13. Choose spells per day (if applicable)
