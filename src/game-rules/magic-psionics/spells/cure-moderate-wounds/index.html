Conjuration (Healing)

Level: Brd 2, Clr 2, Drd 3, Pal 3, Rgr 3

Cleric Domains: Healing 2

This spell functions like cure light wounds, except that it cures 2d8 points of damage +1 point per caster level (maximum +10).
