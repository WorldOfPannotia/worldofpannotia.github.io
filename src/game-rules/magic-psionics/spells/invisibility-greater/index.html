Illusion (Glamer)

Level: Brd 4, Sor/Wiz 4

Components: V, S

Target: You or creature touched

Duration: 1 round/level (D)

Saving Throw: Will negates (harmless)

This spell functions like invisibility, except that it doesn't end if the subject attacks.
